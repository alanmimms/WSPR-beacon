WSPR Protocol Information

WSPR (Weak Signal Propagation Reporter) is a digital mode optimized for weak signal communication.

  • Symbol count: 162 symbols
  • Tone spacing: 1.46 Hz (146 centi-Hz)
  • Symbol period: 0.683 seconds
  • Transmission duration: ~110.6 seconds
  • Bandwidth: ~6 Hz
  • Message types: Callsign, grid locator, and power level

Each symbol represents a frequency offset (0, 1, 2, or 3) from the base frequency, corresponding to 0, 1.46, 2.92, or 4.39 Hz shifts.
